How Study Buddy works

A simple 3-step idea: pair up with a friend, commit to a shared timer, and then reflect on your focus streak together. For more details, open the “How it works” page in the navigation.

Step 1
Pair with a buddy

One student creates a focus room on the website and shares the room code. Their friend joins with the same code from any device.

Step 2
Start the shared timer

Both students agree on a study block, such as 40 minutes. The timer represents a joint promise to avoid phone distractions.

Step 3
Review and grow

After each session, they review the points, warnings and streaks, and slowly build stronger, healthier study habits.

Core design features

These features are conceptual. This prototype focuses on the visual design of the Study Buddy website. The “Features” page describes each benefit in more depth.

01 · Social accountability
Study with someone, not alone

When both students share the same timer, breaking focus affects both partners. This gentle peer pressure reduces “just 2 minutes” of scrolling.

02 · Gentle feedback
Warnings instead of punishment

The system is imagined to give light warnings and small point changes, so students feel guided instead of blocked.

03 · Web-first approach
Runs in any modern browser

No app installation, no sign-up barrier. A link or room code is enough to start a session with a friend.
